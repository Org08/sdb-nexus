
# Pages

## Intro

系統核心內在 `port: 8443` 有數個內建功能頁面.

提供設定用的 `console`, 模擬商店系列功能的 `store`, 方便確認運作的 `monitor`, 還有純文字的後台控制與開發測試介面 `api`. 另外還有各種客製介面.
